# Roadmap

Future improvements and planned features for the dotfiles repository.

## Planned Features

### Installation & Setup

- **User-level installation** - Complete user-level only installation (option to choose between system and user)
- **Package cleanup** - Detect and remove unused packages automatically
- **Automated syncing** - Cronjob to keep package lists and GNOME settings updated

### Configuration Management

- **Asset management** - Drop-in system for fonts, icons, wallpapers with manifests
- **Theme system** - Unified theme management with one-command switching across all apps
- **Browser/editor configs** - Chrome and VS Code settings sync

### Multi-Platform

- **Multi-OS support** - Arch Linux + Hyprland when Ubuntu+GNOME is stable
- **macOS support** - Basic support for development machines

## Ideas (Not Committed)

- Split keyboard configuration management
- Obsidian vault auto-clone during bootstrap
- Automated backup of local customizations
